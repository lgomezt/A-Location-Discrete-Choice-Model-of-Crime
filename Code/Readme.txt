Readme

Data:

- crime_barrio_localidad: Main dataset. Contains information of crimes at the neighborhood leve.
- crime_barrio_dia_localidad: Secondary dataset. Contains information of crimes at the neighborhood-day/night level.

Do-files:

1_descriptive_statistics: Produces descriptives statistics.
2_homogeneous_model: Produces tables and figures related to the estimation of the main model.
3_homogeneous_spatio_temporal: Produces tables and figures related to the estimation of an augmented model that includes day/night as a choice.
4_boostrap: Produces the Standard Errors neede for statistical inference related to counterfactual scenarios.
